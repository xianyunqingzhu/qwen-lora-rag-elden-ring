# 黄金书幽影DLC

## 一、剧情概述

DLC《黄金树幽影》的剧情包含两条核心故事线：

- **明线**：米凯拉的成神之路。米凯拉为改变交界地现状，另起炉灶重新成神，为此抛弃肉体、半身与良知，最终在螺旋塔登神。
- **暗线**：玛丽卡的成神之路。揭示玛丽卡从巫者村出身，通过与角人合作献祭族人登神，再到背叛角人、发动圣战，最终反抗无上意志的完整经历。

此外还包含四条独立支线：

- 梅瑟莫的圣战之路
- 龙族往事
- 癫火之路
- 指头的真相

## 二、世界观背景

### 神祇与王的设定

- **神祇**：类似于宗教的教皇，掌管律法。
- **艾尔登之王**：类似于王国的君主，力量强大。
- **神人**：拥有自己的律法，够资格成为神祇的人。游戏中的神人包括米凯拉、菈妮、玛莲妮亚、宵色眼眸女王等，但真正成为神祇的只有玛丽卡、菈妮和米凯拉。
- **成神方式**：神祇归来需要王的牵引（秘密仪式卷轴记载），登神仪式需献祭大量生命，类似《剑风传奇》中格里菲斯的成神方式。

### 稀人的定义

- 黄金树势力横扫交界地后，其子民对版图之外的类人种族统称为“稀人”。
- 来自永恒之城的黑刀与来自幽影地的玛丽卡同被称为“稀人”。
- 在此定义下，角人对于黄金树子民来说也属于稀人。

### 指头体系

- **无上意志**：最高存在，通过指头传递信息。
- **指头之母**：降临于幽影地，但正式任务前机能损坏，被无上意志抛弃，无法听到无上意志的声音。
- **双指**：由指头之母生出，其中一个双指找到玛丽卡，拥立其为神人。
- **三指**：疑似落在利亚指头遗迹，后来感染癫火。
- 黄金律法实际上是指头之母根据自己的理解和交界地现状编造的，并非无上意志直接传达。

## 三、时间线梳理

### 第一阶段：玛丽卡的成神之路（暗线）

1. **巫者村时期**：玛丽卡出生于巫者村，在祖母雕像前祈祷、许愿、忏悔，离开后再未返回已无人的故乡。
2. **接触指头**：巫者村旁落下双指（狄欧指头遗迹），双指选中玛丽卡成为神人。
3. **与角人合作**：玛丽卡说服角人合作成神，角人修建螺旋塔和神之门。
4. **献祭登神**：神之门由大量类人尸体堆砌而成，献祭了众多巫者和角人。玛丽卡脚踩尸山血海跨过神之门，成为神祇。
5. **背叛角人**：玛丽卡成神后违背盟约，派遣梅瑟莫对角人发起“圣战”，进行种族灭绝。

### 第二阶段：上古之战（黄金树时代前）

1. **封印命定之死**：黑剑击败宵色眼女王，封印命定之死。
2. **巨人战争**：击败火焰巨人与风暴王古龙联军，封印巨人火焰与大锅。荷莱露（葛孚雷）大战风暴王，山妖被策反。
3. **镇压大蛇**：在格密尔火山镇压不死大蛇，开发大蛇狩猎矛。
4. **镇压永恒之城**：诺克斯之民反抗黄金树，打造猎杀指头刀。无上意志降下艾斯提，三座永恒之城沉入地底。

### 第三阶段：长征之战

1. **利耶尼亚战争**：拉达冈崭露头角，卡利亚王室讲和。女王蕾娜拉与拉达冈结合，其妹蕾拉娜被梅瑟莫吸引。
2. **圣战（复仇之战）**：梅瑟莫率军兵临幽影地，建立幽影城，攻破塔之镇，对角人展开种族屠杀。玛丽卡用迷雾将幽影地隐去，抹除相关历史记载。
3. **征服宁格姆福与啜泣半岛**：葛孚雷在此期间完成征服。
4. **生育子嗣**：玛丽卡与葛孚雷生下黄金长子葛德温、恶兆双子蒙葛特和蒙格。长角诅咒源于屠杀角人后黄金树熔炉纳入大量角人怨魂。
5. **褪色者诞生**：女王剥夺葛孚雷及王师的赐福，发配到交界地之外，成为第一波褪色者。

### 第四阶段：黄金树丰饶时代

1. **拉达冈继任**：拉达冈被调回王城继任艾尔登之王，与玛丽卡生下米凯拉和玛莲妮亚（双胞胎均有缺陷）。
2. **古龙战争**：古龙袭击王城，葛德温战功显赫，赢得古龙尊重。
3. **碎星战争**：拉塔恩在盖利德抵御群星入侵，封印星星，获称碎星将军。少年拉塔恩与米凯拉做下成王约定。

### 第五阶段：黄金树体系崩塌

1. **阴谋之夜**：玛丽卡与菈妮、拉卡德合作，偷走命定之死，杀死葛德温的灵魂，菈妮杀死自己的肉体摆脱双指。
2. **碎环**：玛丽卡砸碎艾尔登法环，大卢恩散落，半神争夺引发破碎战争。
3. **神人被囚**：无上意志囚禁玛丽卡于黄金树内，用拒绝的刺封闭入口。
4. **引导褪色者**：玛丽卡派出女儿梅琳娜引导褪色者成王，打破黄金树律法。

### 第六阶段：米凯拉的成神之路（明线）

1. **放弃黄金律法**：米凯拉认识到玛丽卡路线行不通，决定另起炉灶重新成神。
2. **杀死拉塔恩**：拉塔恩拒绝成为其王，米凯拉指使玛莲妮亚攻打拉塔恩。
3. **制作王**：米凯拉自愿被蒙格掳走，反杀蒙格，将拉塔恩的灵魂装入蒙格的尸体，为自己创造王。
4. **抛弃自身**：一路抛洒肉身、大卢恩、半身托莉娜（爱与恐惧），抵达螺旋塔。
5. **登神**：在螺旋塔顶成神，被褪色者击败。

## 五、关键人物

### 玛丽卡

- **身份**：巫者村出身的巫者，神人，后成为黄金树体系唯一成神者，永恒女王。
- **成神经历**：被双指选中后与角人合作，献祭大量巫者与角人登神，背叛角人，发动圣战灭族。
- **关键事件**：碎环破坏黄金树体系，策划阴谋之夜，引导褪色者回归。
- **特征**：成神后仍在巫者村留下编发与黄金树，显示“人心未泯”。后期看破黄金律法虚伪，反叛无上意志。

### 米凯拉

- **身份**：玛丽卡与拉达冈自交之子，永远幼体，神人。
- **特征**：雌雄同体（另一半为托莉娜），充满爱和包容之心，积极探索黄金律法。
- **成神计划**：舍弃肉体、半身托莉娜（爱与恐惧），将拉塔恩灵魂装入蒙格尸体制成王，在螺旋塔成神。
- **魅惑本质**：强制他人“相互理解”，被魅惑者停止斗争并保留记忆。大卢恩破碎后效果解除。
- **结局**：被褪色者击败。

### 梅瑟莫

- **身份**：玛丽卡之子（生父不明），半神，梅琳娜的兄长。
- **特征**：红发，体内有恶之蛇力量，拥有烧树火种。
- **圣战**：主动承担不名誉的圣战总指挥，对角人进行种族灭绝。建立物种保存库与巫者关怀设施。
- **结局**：圣战接近全胜时接到碎环消息，镇守幽影城直至被击败。

### 梅琳娜

- **身份**：玛丽卡之女，梅瑟莫之妹。
- **特征**：常闭一只眼（继承命定之死力量），拥有烧树火种（继承巨人火焰）。
- **任务**：引导褪色者成王，为烧毁拒绝的刺提供火种。

### 菈妮

- **身份**：拉达冈与蕾娜拉之女，神人。
- **目标**：建立没有律法的星星时代。
- **关键事件**：参与阴谋之夜杀死自己的肉体，摆脱双指控制；派遣褪色者获取猎杀指头刀杀死双指。

### 拉塔恩

- **身份**：拉达冈与蕾娜拉之子，碎星将军。
- **特征**：精通重力魔法，崇拜葛孚雷。少年时与米凯拉约定成王，后拒绝赴约。
- **结局**：在碎星战争被玛莲妮亚攻击，肉体死亡，灵魂被米凯拉装入蒙格尸体成为“约定之王”。

### 托莉娜

- **身份**：米凯拉的半身，代表其爱与恐惧。
- **特征**：拥有独立人格，反对米凯拉成神，称成神是米凯拉的牢笼，指引玩家杀死米凯拉助其解脱。

### 角人

- **身份**：幽影地原住民，崇拜熔炉，信仰螺旋塔。
- **与玛丽卡关系**：曾与玛丽卡合作建塔助其成神，被背叛后遭灭族。
- **制品**：将巫者切碎装入壶中，视为使其“从罪人变为好人”的仪式。

## 六、关键事件

### 玛丽卡登神

- **地点**：幽影地螺旋塔神之门。
- **过程**：献祭大量巫者与角人，以尸山血海为祭品跨过神之门。
- **结果**：玛丽卡成为唯一跨过神之门成神的神人。

### 圣战（角人灭族）

- **发起者**：玛丽卡。
- **执行者**：梅瑟莫率圣战军。
- **方式**：火焰攻击为主，柳条人、火焰枪等手段。
- **范围**：角人、亚人、混种均在圣战范围内。
- **结果**：角人几乎灭族，幽影地被迷雾隐藏，梅瑟莫被藏匿于历史记载之外。

### 阴谋之夜

- **参与者**：玛丽卡、菈妮、拉卡德。
- **执行者**：黑刀刺客（菈妮的双指私兵）。
- **手段**：偷取黑剑封印的命定之死制作黑刀。
- **结果**：葛德温灵魂死亡，菈妮肉体死亡，为黄金树体系崩塌埋下伏笔。

### 碎环

- **参与者**：玛丽卡。
- **过程**：玛丽卡砸碎艾尔登法环。
- **结果**：大卢恩散落，破碎战争爆发，黄金树王朝支离破碎。玛丽卡被无上意志囚禁。

### 米凯拉登神

- **地点**：螺旋塔顶。
- **过程**：抛弃大卢恩、肉体、半身托莉娜后登神。
- **结果**：被褪色者击败。

## 七、重要概念与设定

### 壶与巫者

- **角人的行为**：角人认为将巫者切碎装入壶中是“让罪人变为好人”，可能源于认为巫者是“苍蝇病”的罪魁祸首。
- **监牢中的壶**：贝瑞特监牢、波尼监牢有大量巫者尸体和壶。
- **玛丽卡进壶**：推测玛丽卡可能曾被装入壶中，DLC画面显示玛丽卡从类似壶的内容物中抓出法环雏形。

### 成神的代价

- **祭品要求**：神之门需要献祭大量生命（神之门由类人尸体堆砌而成）。
- **人性抛弃**：米凯拉为成神抛弃肉体、半身、良知；玛丽卡献祭族人。
- **诅咒伴随**：玛丽卡屠杀角人后被诅咒，所生孩子带角（恶兆蒙格与蒙葛特）。米凯拉与玛莲妮亚亦生而有缺陷。

### 魅惑

- **机制**：强制被魅惑者“相互理解”，停止斗争，保留魅惑期间记忆。
- **范围**：米凯拉通过魅惑使多个阵营在圣树共同生活，被压制者包括蕾妲（多疑被压制）。

### 拒绝的刺

- **作用**：封闭登神塔，阻止他人成神；封印黄金树入口，阻止他人成王。
- **破解方式**：需要火种烧毁（梅瑟莫、梅琳娜均拥有此能力）。

### 癫火

- **特点**：随时间泄漏污染，接触者变为癫火怪物。受赐癫火者最终将成为癫火之王或只剩脊柱。
- **故事**：米德拉接触癫火后激烈拒绝成为癫火之王，被永罚大剑穿刺镇压。

### 指头的真相

- **通讯链路**：无上意志→指头之母→双指→神王组合。
- **实际状况**：指头之母被无上意志抛弃，黄金律法为指头之母自行编造。
- **尤弥尔的发现**：调查发现指头之母已无法联系无上意志，试图取而代之，最终发疯。

## 八、争议与不同说法

### 争议内容：巫者与角人是否同族

- **说法一**：巫者与角人是同族，角人猎巫是同族相残，使故事更具悲剧性。
- **说法二**：巫者与角人为不同种族，角人强制将巫者装壶。
- **现状**：官方未明确给出设定。

### 争议内容：玛丽卡进壶的时间

- **说法一**：玛丽卡先被装壶，后被双指找到成为神人。
- **说法二**：玛丽卡先被双指选中，后因某些原因被装壶。
- **现状**：原文未明确，仅为玩家推测。

### 争议内容：玛丽卡反叛黄金树律法的原因

- **推测一**：完成复仇后，成神前抛弃的人性觉醒，看尽战争残酷后对黄金律法产生怀疑。
- **推测二**：玛丽卡想对抗强加给自己的命运，获得真正的自由。
- **现状**：原文称“目前还不明了，希望环学家探明”。

### 社区推测：梅瑟莫与梅琳娜的父亲与力量来源

- **父亲**：推测为玛丽卡自我生育（无父亲），幽影城保藏库设有三把空座椅对应玛丽卡及兄妹。
- **力量来源**：
  - 梅瑟莫体内恶之蛇力量可能来自火山大蛇。
  - 梅琳娜命定之死力量来自宵色眼女王，火种力量来自火焰巨人。
  - 红发特征可能源自火焰巨人。

### 社区推测：玛丽卡参与阴谋之夜的证据

- **依据**：黑剑追忆描述“玛丽卡希望她的影子成为命定之死的封印，然而最后，她却背叛了影子”，推测背叛指协助偷走命定之死。
- **执行分析**：玛丽卡、菈妮、拉卡德三人合作策划阴谋之夜。

## 九、梅瑟莫相关剧情整理

### 基本信息

- **身份**：玛丽卡之子，半神，梅琳娜兄长，圣战军总指挥。
- **特征**：红发，体内有恶之蛇力量，拥有烧树火种。
- **所在区域**：幽影城。

### 圣战经历

1. 主动承担不名誉的圣战总指挥。
2. 率火焰骑士和蕾拉娜进攻幽影地，建立幽影城。
3. 攻破塔之镇，对角人进行种族灭绝式屠杀。
4. 扫荡各遗迹，亚人在绝望中接触真实之母。
5. 击败角人后救出罐中巫者，在幽影城地下建立临终关怀设施。
6. 建立物种保存库，焦黑角人在此充当学者。
7. 接近全胜时接到玛莉卡碎环消息，维持战线不再推进。

### 相关人物

- **蕾拉娜**：满月女王之妹，卡利亚驻幽影地总指挥，因爱慕梅瑟莫投身圣战。
- **火焰骑士**：梅瑟莫直属军队。
- **玛丽卡**：母亲，圣战发起者。

### 评价

- 在所有半神中“绝对不是该死的那一类”，圣战主要责任在玛丽卡。
- 因拥有火种能力而“怀璧其罪”。

## 十、米凯拉成神流程整理

1. **认识黄金律法局限**：通过探索和滋养圣树经历，明白必须自己成神。
2. **约定之王拒绝**：少年时与拉塔恩做约定，拉塔恩拒绝赴约。
3. **杀死拉塔恩**：指使妹妹玛莲妮亚攻打拉塔恩，杀死其肉体。
4. **魅惑蒙格**：自愿被蒙格掳走，反魅惑并杀死蒙格，夺取其尸体。
5. **制作王**：将拉塔恩灵魂装入蒙格尸体，创造“约定之王”。
6. **抛弃自身**：
   - 抛弃神人躯体
   - 抛弃大卢恩
   - 抛弃半身托莉娜（爱与恐惧）
7. **抵达螺旋塔**：一路抛洒肉身抵达成神地点。
8. **登神**：在螺旋塔顶成神。
9. **被褪色者击败**：成神后扫平眼前褪色者时被击败。

## 十四、龙族往事

1. 狂龙贝勒向龙王挑战，咬掉龙王两个头。
2. 贝勒被咬掉双脚、双翼、双手，仅剩半只手臂，流放至幽影地尖刺山。
3. 尖刺山旁可见大量龙尸。
4. 龙王留下龙巫女为首的力量看守贝勒。

## 十五、癫火支线

1. 角人学者米德拉在幽谷中拥有大书库府邸。
2. 接触到手持癫火之王脊柱手杖的娜娜亚并爱上她。
3. 米德拉开始研究癫火并沾染癫火，二人压制癫火不愿焚毁世界。
4. 癫火蔓延污染山谷，角人发现后冲入府邸杀尽众人。
5. 米德拉激烈拒绝成为癫火之王，被永罚大剑从头顶穿刺镇压。
6. 癫火随时间泄漏，山谷成为癫火信徒圣地。

## 十六、指头支线

1. 指头之母降临幽影地后机能损坏，被无上意志抛弃。
2. 指头之母生下大量指头，双指找到玛丽卡拥立其成神。
3. 黄金律法实为指头之母自行编造。
4. 诺克隆恩发明指头刀后，无上意志降下流星镇压。
5. 玛丽卡发现黄金律法问题，不再相信双指，砸环。
6. 魔法教授尤弥尔调查发现指头之母秘密，试图取而代之，在接触禁忌知识后发疯长满手指。

## 十七、总结

玛丽卡的故事主线：

```
经历磨难 → 渴求力量 → 借助无上意志成神 → 为无上意志代言 → 
报私仇 → 看破黄金律法虚伪 → 解构自我反叛无上意志 → 
策划颠覆行动（驱逐褪色者、阴谋之夜、碎环） → 
引导褪色者打破律法
```

米凯拉的故事主线：

```
认识黄金律法局限 → 与拉塔恩约定 → 拉塔恩拒绝 → 
杀死拉塔恩 → 夺取蒙格尸体制作王 → 
抛弃肉体与半身 → 登神 → 被褪色者击败
