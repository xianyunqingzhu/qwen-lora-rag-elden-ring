# 呆萌少女菈雅 (Zorayas)

## 一、人物概述

菈雅，真名泽菈雅斯 (Zorayas)，是《艾尔登法环》中火山官邸的 NPC。她以呆萌、纯真、善良的性格深受玩家喜爱，日常形态为人类少女，真实形态为蛇人。她在游戏中担任火山官邸的“HR”，负责在交界地招募有资质的褪色者加入阵营。她的故事围绕身份认同、身世真相与母女情感展开，是游戏中最令人动容的支线之一。

## 二、身份与背景

### 2.1 身份信息

- **化名**：菈雅 (Rya)
- **真名**：泽菈雅斯 (Zorayas)
- **种族**：蛇人
- **日常形态**：人类少女（为方便招募而伪装）
- **真实形态**：身穿衣物的蛇形生物
- **职位**：火山官邸招募者（侦察兵）
- **养母**：塔尼斯 (Tanith)，火山官邸的代理主人
- **生母**：被拉卡德融合的上古大蛇（通过仪式诞下蛇人）
- **生父（广义来源）**：亵渎君王拉卡德 (Rykard) 与上古大蛇融合后的存在

### 2.2 蛇人起源

- 蛇人的诞生源于一场可怖的仪式。拉卡德为获得亵渎神祇的力量，自愿被格米尔火山的上古大蛇吞噬。拉卡德与被吞噬者的意志融合后，诞生了一群蛇人。菈雅正是这些蛇人中的一员。
- 仪式前，拉卡德曾赠予宠爱的舞女塔尼斯一瓶“遗忘秘药”，希望她忘记这场恐怖的仪式。塔尼斯拒绝饮下，从诞生的众多蛇人中选择了一只小蛇，带回去当作亲生女儿抚养，取名为泽菈雅斯。

## 三、生平经历

### 3.1 成长环境

菈雅被塔尼斯抚养长大。塔尼斯深知女儿天真单纯，始终为她营造出一个充满尊严与自豪感的成长环境。塔尼斯告诉菈雅，她是“由一位伟大君王的恩典所生”，这让菈雅对自己的蛇人形态怀有骄傲。菈雅也深爱并敬仰着养母，连发型都与塔尼斯梳成同款盘发。在火山官邸中，由于外形可能影响招募工作，她通常伪装成人类少女示人。

### 3.2 初次相遇：项链事件

- **地点**：利耶尼亚湖区的凉亭
- **触发条件**：玩家进入利耶尼亚后，在凉亭附近听到少女的呼唤。
- **剧情流程**：
  1. 菈雅向褪色者求助，称自己的项链被一名流氓抢走。
  2. 褪色者前往煮虾哥所在的破屋，煮虾哥索要 1000 卢恩才肯交出项链（推测菈雅曾饿倒被虾哥所救，用项链抵了饭钱）。
  3. 褪色者交付卢恩，取回项链。
  4. 将项链还给菈雅后，菈雅自我介绍，说明自己是火山官邸的仆人，负责招募英雄，并赠予玩家“火山官邸邀请函”。
  5. 菈雅指引褪色者前往亚坛高原，期待再次相见。

### 3.3 正式入队：传送至火山官邸

- **地点**：亚坛高原，迪可达斯大升降机出口附近
- **剧情流程**：
  1. 玩家到达亚坛高原后，在赐福旁再次见到菈雅。
  2. 菈雅表示褪色者已通过最终面试，牵起玩家的手，直接将玩家传送至火山官邸。
  3. 在火山官邸，塔尼斯作为总经理，向褪色者介绍火山官邸的核心理念：反抗黄金律法，犯下叛律的亵渎之举，并许诺分配房间。
  4. 褪色者可选择加入火山官邸，成为“叛逆者”，并接到猎杀其他褪色者的委托。

### 3.4 身份初显：蛇人形态暴露

- **触发条件**：加入火山官邸并完成初步任务后，与菈雅交谈。她提及自己最近听到类似呼吸声与鳞片摩擦的奇怪声音，怀疑自己太累。之后她回房休息，玩家前往她的房间。
- **剧情流程**：
  1. 推门进入房间，发现一个身穿菈雅衣物的蛇形生物，这正是菈雅的真实形态。
  2. 菈雅惊慌失措，承认自己真名是泽菈雅斯，蛇形才是她的真面目。她解释自己为远离世人歧视而伪装成人类。
  3. 菈雅强调自己为这副形态骄傲，因为母亲（塔尼斯）珍视她的样子，并坚称自己是由“伟大君王的恩典”所生。
  4. 褪色者向塔尼斯询问此事，塔尼斯没有透露更多信息，只恳请褪色者好好对待菈雅。

### 3.5 寻求真相：发现官邸秘密

- **触发条件**：完成第二个刺杀任务后，与菈雅多次交谈。
- **剧情流程**：
  1. 菈雅向褪色者透露，夜间看到蛇形黑影进入隔壁房间后再未出来，怀疑官邸藏有母亲隐瞒的秘密。
  2. 玩家从菈雅隔壁房间发现隐藏通道，一路深入，穿过密室到达火山官邸后院。
  3. 后院充满了与菈雅形态相似的蛇人、被拷问的白金之子、铁处女、人类尸体等，宛如人间炼狱。
  4. 褪色者在教堂祭台找到“蛇的羊膜”，将其带回给菈雅。
  5. 菈雅认出这是自己生母的一部分，记忆涌现。她感谢褪色者，并决定不再恐惧，要亲自寻找真相，弄清楚自己如何出生、如何遇到塔尼斯。随后，她独自走入隐藏通道。

### 3.6 崩溃与请求：身世真相揭晓

- **触发条件**：菈雅进入隐藏通道后，玩家在后院某狭小房间内找到她。
- **剧情流程**：
  1. 菈雅已得知全部真相：自己并非恩惠所生，而是源自亵渎君王拉卡德与大蛇融合的仪式。她是塔尼斯从上万条蛇中挑选出来培养的。无论在人类还是蛇人眼中，她都是不被接受的存在。
  2. 菈雅陷入极度自卑与绝望，请求褪色者杀死她，让她从“被诅咒的躯壳”中解脱。
  3. 塔尼斯找到褪色者，作为母亲恳求，如果菈雅因真相而痛苦，请褪色者给她服下“遗忘秘药”，让她忘记一切。塔尼斯承认这是一种不尊重，但也别无选择。
  4. 褪色者面临抉择：直接杀死菈雅、让她服下秘药、或者暂时离开给她时间。

### 3.7 告别与旅行：菈雅的结局

- **剧情流程**：
  1. 褪色者暂时离开，前去击败亵渎君王拉卡德（使用大蛇狩猎矛）。
  2. 击败拉卡德后，回到菈雅所在的小房间，发现她已不在，只留下一封信和生母的雕像。
  3. **信件内容**：菈雅决定出门旅行，希望有朝一日能以塔尼斯女儿的身份，继承母亲的壮志。她感谢褪色者的温暖与严格，称褪色者为“我的英雄”，并期待再会。

## 四、关键事件

1. **利耶尼亚项链事件**：初次见面，褪色者帮她从煮虾哥处取回项链。菈雅提供火山官邸邀请函。
2. **亚坛高原引路**：再次见面，传送褪色者至火山官邸，正式开启火山支线。
3. **真身暴露**：褪色者无意发现菈雅蛇人形态，菈雅坦承身份，建立深厚信任。
4. **秘密发现**：菈雅感到不安，褪色者调查官邸后院，发现蛇人炼狱并带回“蛇的羊膜”。
5. **身世冲击**：得知自己是亵渎仪式产物的真相后，菈雅精神崩溃并请求死亡。
6. **英雄的抉择**：褪色者未杀她亦未强行喂药，选择给菈雅时间。
7. **菈雅的旅行**：菈雅自己走出阴霾，留下信件独自踏上旅程，寻找自我意义。

## 五、人物关系

- **养母塔尼斯**：菈雅最敬仰与深爱的人。塔尼斯虽有所隐瞒，但对菈雅的爱真诚真挚。菈雅对自己的身份认同很大程度上源于塔尼斯的肯定。
- **生父/来源 亵渎君王拉卡德**：火山官邸的幕后主人，与上古大蛇融合，是菈雅等蛇人诞生之源。菈雅在得知真相前，可能曾视其为“伟大君王”。
- **褪色者 (主角)**：被菈雅视为可以信任的英雄。褪色者引导她发现了真相，也给了她面对现实的勇气和时间。菈雅在信件中称其为“我的英雄”。
- **煮虾哥 (布莱德)**：与菈雅有过尴尬的初遇（因吃虾付不起钱被拿走项链），但本质不坏。

## 六、相关势力

- **火山官邸**：菈雅效忠的阵营，目标是反抗黄金律法，犯下叛律之举。表面上是招募英雄的反抗军，内部隐藏着拉卡德吞噬强者、制造蛇人等黑暗秘密。

## 七、相关地点

- **利耶尼亚湖 (Liurnia of the Lakes)**：凉亭，初次相遇地点。
- **煮虾哥的破屋 (Boilprawn Shack)**：找回项链的地点。
- **亚坛高原 (Altus Plateau)**：再次见到菈雅，并通过她传送至火山官邸的地点。
- **火山官邸 (Volcano Manor)**：菈雅的主要活动区域，包括会客厅、她的房间、隐藏通道及后院。

## 八、相关道具与奖励

- **火山官邸邀请函**：完成项链任务后获得，指引前往亚坛高原。
- **蛇的羊膜**：在火山官邸后院教堂祭台找到，用于推动菈雅支线。
- **遗忘秘药**：塔尼斯在菈雅得知真相后给予褪色者，可以选择给菈雅服下以消除记忆。如未使用，可留作纪念品。
- **火山官邸支线奖励**：完成菈雅支线并不影响获得猎杀委托的装备，但菈雅的故事走向完全取决于玩家选择（给不给药/杀不杀她）。后续她留下的信和生母雕像也是特殊剧情物品。

## 九、争议与不同说法

### 9.1 蛇人起源细节

- **原作设定**：明确提到蛇人由“拉卡德与大蛇融合后的仪式”诞生，菈雅是其中之一。塔尼斯从蛇群中挑选了她。
- **不同说法**：有资料提及“格米尔火山的老蛇吞噬了半神产下蛇人”，此处“半神”即指拉卡德。两种说法本质一致，只是描述视角不同。

### 9.2 “菈雅是否完全不知官邸黑暗面”

- **设定共识**：多数文本指向菈雅在成长过程中被塔尼斯完全保护，对官邸的骸骨、拷问等黑暗面在褪色者到来前确实一无所知。她的纯真是真实的，并非伪装。
- **细节讨论**：菈雅曾听到“呼吸声和鳞片摩擦声”，说明她潜意识中已经隐约感到不安，但被训练成“觉得自己太累”，直到褪色者介入，她才逐步正视异样。

## 十、补充说明

- **菈雅的纯真性格**：即使在最崩溃的时刻，她也没有怨恨塔尼斯，反而希望“有一天能从心底叫她一声母亲”。她旅行前留下的话，字里行间依然是对母亲的爱和对未来的期许。
- **发型细节**：菈雅的发型和塔尼斯一致，象征着她对养母的崇拜与深深的归属感。
- **英雄的寓意**：菈雅的故事中，“英雄”褪色者并没有用遗忘或毁灭的方式解决问题，而是给了她时间和勇气，让这只小蛇自己去面对残酷真相，走向真正的成长。这恰恰呼应了菈雅最初招募时寻找的“英勇却蒙尘的人”的标准。
