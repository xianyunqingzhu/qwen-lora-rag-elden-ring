# 碎星将军拉塔恩

## 一、人物概述

拉塔恩是黄金一族半神，父亲为红发英雄拉达刚，母亲为满月女王蕾娜拉，继承了父亲的一头红发，并以此傲。他自幼崇拜初代艾尔登之王葛弗雷，立志成为黄金树的狮子与英雄。拉塔恩重情重义，为让爱马承载自己日渐魁梧的身躯，前往魔法重镇瑟利亚学习重力魔法，后以重力魔法击碎流星、封印群星运转，获称“碎星将军”。他在盖立德建立红狮子城，组建“永无懦夫”的红狮子军团。破碎战争中，拉塔恩与女武神玛连尼亚于盖立德决战，玛连尼亚释放猩红腐败，盖立德沦为腐化之地，拉塔恩亦受感染。战后，他在理智尚存时建立火墙阻止腐败扩散，安葬敌我将士，后自我放逐于荒漠沙丘。最终，其挚友杰连为成全他“战死沙场”的荣誉，召开战斗祭典，拉塔恩在众多战士围攻下力竭战死，结束波澜壮阔的一生。

## 二、身份与背景

- **出身**：黄金王室拉达刚与满月女王蕾娜拉的长子。
- **血统**：父系为黄金树神格，母系为卡利亚王室。
- **外貌特征**：继承父亲的红发，视之为英雄的象征。
- **信仰与志向**：崇拜初代艾尔登之王葛弗雷，梦想成为“黄金树的狮子”，化身战王的利爪。
- **封号**：碎星将军。
- **封地**：盖立德。
- **居城**：红狮子城。
- **麾下军队**：红狮子军团，被誉为“永无懦夫”。

## 三、生平经历

### 1. 出生与童年

- 拉达刚与满月女王蕾娜拉在利耶尼亚的教堂缔结契约后成婚，生下三位半神后代：拉塔恩、拉尼、拉卡德。
- 拉塔恩自幼崇拜初代艾尔登之王葛弗雷与其宰相色洛休的传说，梦想成为葛弗雷般的英雄。
- 童年接受严苛战斗训练，少年时期已展现出英雄气概，吸引了一批忠诚死士。

### 2. 学习重力魔法与“碎星”由来

- 随着拉塔恩身躯日渐魁梧，自幼跟随他的瘦马难以承载。
- 拉塔恩不愿抛弃战友，与亲信红狮子骑士奥加一同前往魔法重镇瑟利亚，学习重力魔法。
- 瑟利亚为他引荐上古种族白王（随陨石降落、精通重力魔法）。凭借母系满月血统带来的天赋，拉塔恩很快掌握重力魔法，并将重力徽章刻于大剑之上。
- 一颗流星将坠，其中藏有坠星兽。拉塔恩以刻有重力符文的大剑将其击碎，封印于瑟利亚矿坑深处。“碎星大剑”与“碎星将军”称号由此而来。
- 为防止星空威胁再次降临，拉塔恩以重力魔法彻底封印群星运转，将自身命运与星空流转绑定。

### 3. 家族变故与建立红狮子城

- 拉塔恩崇拜的葛弗雷及其麾下战士被女神玛丽卡剥夺赐福、放逐出交界地。
- 此后父亲拉达刚离开蕾娜拉，返回王城罗德尔与玛丽卡成婚，成为第二任艾尔登之王。
- 母亲蕾娜拉因此一蹶不振，被雷亚卢卡利亚魔法学院剥夺权力并囚禁。
- 拉塔恩成为玛丽卡继子，以黄金一族外戚身份获封盖立德。
- 他在封地建立城塞，命名为红狮子城，创建红狮子军团。
- 曾到卡利亚王室做客的骑士杰连，在拉塔恩建立狮子城后，作为客将加入麾下，两人建立深厚战友情。

### 4. 破碎战争与进军王城

- 艾尔登法环破碎，拉达刚消失。拉塔恩对玛丽卡放逐葛弗雷本就心存质疑，父亲的消失令他决心行动。
- （推测）拉塔恩并非为争夺大卢恩，而是想查明真相、前往王城“清君侧”。
- 拉塔恩集结红狮子大军，兵临王城罗德尔。
- 失去王与神的王城无力防御，黄金后裔葛瑞克混在女眷中连夜出逃。
- 被视为“恶兆”的蒙格特挺身守卫王城。拉塔恩最终停止进攻，退兵返回盖立德。

### 5. 盖立德决战：拉塔恩 vs 玛连尼亚

- 女武神玛连尼亚率领尊府骑士大军南下，一路所向披靡，击败葛瑞克并迫其臣服。
- 拉塔恩命老将欧尼尔集结红狮子军团，在盖立德严阵以待。
- 两军交战，红狮子与尊府骑士势均力敌。玛连尼亚直取拉塔恩。
- 拉塔恩利用重力魔法使碎星大剑攻击更为沉重，逐渐压制玛连尼亚，击落其假肢。
- 玛连尼亚被逼至绝境，拔出体内抑制猩红腐败的金针。
- 爱奥尼亚花在战场绽放，猩红腐败瞬间蔓延整个盖立德，腐化大地与生灵。
- 拉塔恩正面受到腐败冲击，神志开始受损，但仍保持理智未被击倒。
- 玛连尼亚释放力量后陷入沉睡，由尊府骑士芬雷抱起率残部撤退。

### 6. 战后坚守与自我放逐

- 拉塔恩率领未受感染的士兵建立“星火高墙”，阻止腐败继续扩散。
- 老将欧尼尔深陷腐败腹地未及撤退，仍指挥灵魂士兵坚守。
- 拉塔恩带领部下打扫战场，修建“英灵地下墓穴”，将战死的红狮子骑士与尊府骑士一同厚葬。
- 因法环破碎与死亡符文失窃，双方战死者灵魂无法回归黄金树，在墓穴中永无休止地战斗。
- 红狮子军团烧毁左胸上的徽章，告别故乡，誓死留在此地终生抑制腐败。
- 拉塔恩自知被腐败侵蚀日渐失控，独自前往战场中央的荒漠沙丘，自我放逐，将对外界伤害降至最低。
- 即使失去理智，他仍持续以重力魔法封锁星空，阻止群星坠落。
- （一种说法）他甚至自断双腿，以防自己跑出盖立德祸乱世间。

### 7. 战斗祭典与结局

- 客将杰连遵守与拉塔恩“彼此战死沙场”的誓言，为让将军荣誉战死，在红狮子城召开战斗祭典。
- 众多战士（包括半狼布莱泽、战士壶亚历山大等）齐聚，与已失去理智的拉塔恩战斗。
- 拉塔恩在战斗中仍以重力魔法保护胯下瘦马。他力竭时召唤陨石攻击，最终被击败。
- 拉塔恩战死后，封印解除，一颗巨大流星坠落，揭示他至死仍在封锁星空威胁。
- 杰连接连见将军死得其所，覆面痛哭，随后离开盖立德继续流浪。

## 四、关键事件

1. **习得重力魔法**：为爱马前往瑟利亚，师从白王。
2. **击碎流星，封印群星**：获封“碎星将军”，绑定命运与星空。
3. **进军王城**：破碎战争初期，为查明父亲下落兵临罗德尔，与蒙格特对峙后退兵。
4. **盖立德决战**：与玛连尼亚爆发碎片战争中最为惨烈的战斗。玛连尼亚释放猩红腐败，盖立德沦为腐地。
5. **战后坚守**：建立火墙，修建墓穴，军团留驻抑制腐败。
6. **自我放逐**：为避免伤害他人，独自在荒漠沙丘等待死亡。
7. **战斗祭典**：杰连召集英雄，拉塔恩在战斗中荣誉战死。

- **杰连（挚友、客将）**：流浪骑士，曾到卡利亚王室做客，后加入拉塔恩麾下，主持战斗祭典。

- **玛连尼亚（宿敌）**：女武神，在盖立德决战中释放猩红腐败，改变拉塔恩命运。

- **红狮子军团**：拉塔恩创建，被称为“永无懦夫”，战后全员留驻盖立德抑制腐败。
- **瑟利亚魔法镇**：学习重力魔法之地，结缘深厚。

## 八、相关Boss / 道具

- **Boss战：碎星将军拉塔恩**：在红狮子城战斗祭典中进行，具有极高表演性质。战时仍以重力魔法保护战马，末期可召唤陨石。
- **碎星大剑**：刻有重力徽章的大剑，曾击碎流星。
- **拉塔恩的大卢恩**：其死后可由褪色者获取。
- **狮子铠甲**：拉塔恩自己设计的铠甲。
- **瘦马**：拉塔恩自幼的伙伴，拉塔恩习得重力魔法以免压垮它。

## 九、争议与不同说法

### 1. “碎星”称号的真正由来

- **直接关联说法**：拉塔恩击碎了即将砸向瑟利亚的流星（其内藏有坠星兽），并封印于矿坑，以此得名。
- **Boss战后显现说法**：部分观点认为他封印的一直是死后才坠落的那颗巨大流星，与矿坑内的艾丝提无关。
- **折中推测**：若他从未击碎过星星，便不会有碎星称号。矿坑中的坠星兽与封印的巨大流星，可能同为他封印群星的一部分。

### 2. 学习重力魔法的初衷是否为纯粹巧合

- **已知事实**：拉塔恩为减轻爱马负担而学习重力魔法，后却以此封印群星。
- **推测**：这可能并非巧合，但目前没有确凿证据表明他学习之前便有此目的。

### 3. 拉塔恩是否曾返回母亲处

- **推测**：拉塔恩身处盖立德，封地与卡利亚王室之间隔着史东薇尔城，且必经之路可能设有伏击。蕾娜拉被囚禁的消息外人知之甚少。很大概率拉塔恩并未返回。

### 4. 拉塔恩在王城退兵的真实原因

- **实力说法**：蒙格特实力更胜一筹，迫使拉塔恩退兵。
- **觉悟说法**：蒙格特守卫黄金树的决心与拉塔恩利益一致，拉塔恩意识到对方并非敌人而自主退兵。

### 5. 拉塔恩自我放逐后的极端措施

- **自断双腿说法**：存在“拉塔恩自断双腿防止自己跑出盖立德”的说法，但此细节可能源于玩家社区解读或推测，游戏中直接证据有待确认。
